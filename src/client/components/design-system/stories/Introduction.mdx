# JobDash Design System

A custom Tailwind v4 design system built for a minimal-warm aesthetic:
glass surfaces, rounded corners, warm palette, native dark-mode.

No external UI library. All primitives live under
`src/client/components/design-system/` and are framework-agnostic
inside the React + Vite client.

## Design tokens

Defined in [`src/client/index.css`](../../../index.css) via Tailwind v4
`@theme`.

### Glass surface utility

```css
.glass {
  background: rgba(255, 255, 255, 0.55);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(255, 255, 255, 0.5);
}
.dark .glass {
  background: rgba(39, 39, 42, 0.65);
  border-color: rgba(63, 63, 70, 0.5);
}
```

## Primitive catalog

Explore each primitive in the sidebar. All primitives:

- accept a `variant` (usually `"glass"` or `"raised"`)
- accept a `size` where applicable (`"sm" | "md"` or `"sm" | "md" | "lg"`)
- use design tokens instead of hard-coded colors
- render identical in both light and dark mode (toolbar toggle above)

## When to reach for what

| Need | Use |
|------|-----|
| Text / email / password input | `Input` |
| Numeric with optional prefix / suffix | `NumberField` |
| Date, datetime, time | `DateField` |
| Single-select dropdown | `Select` |
| Plain multi-line text | `Textarea` |
| Markdown notes (with preview) | `MarkdownField` |
| Primary / secondary / ghost action | `Button` |
| Status pill | `Badge` |
| Surface / panel | `Card` |
| Overlay dialog | `Modal` |
| Preset / tab filters | `FilterChips` |
| Scrollable table column header | `ColumnHeader` / `SortableHeader` |
| Global command / search input | `SearchBar` |
| Detail-page tab row | `TabBar` |
