# JobDash Design System · v1

A custom Tailwind v4 design system — warm, glass-backed, with an amber
accent and JetBrains Mono labels. Inspired by the JobDash design brief
in `logo/JobDash Design System.html`.

No external UI library. All primitives live under
`src/client/components/design-system/` and consume the CSS tokens
defined in `src/client/index.css`.

## Foundations

### Typography

Three families, each with a clear role:

| Family | Role | Usage |
|--------|------|-------|
| **System sans** (`var(--font-sans)`) | Body copy | Paragraphs, table cells, field values |
| **Archivo** (`var(--font-display)`) | Display / headings | Section titles, page headers, sticky-note headlines |
| **JetBrains Mono** (`var(--font-mono)`) | Labels + data | Field labels (uppercase), mono badges, tabular numerals, keyboard hints |

The app-wide `.ds-label` utility encodes the canonical label treatment:
10px JetBrains Mono 600-weight uppercase with 0.08em tracking. Reach
for it whenever you'd otherwise write 4 equivalent Tailwind classes.

### Glass + dot-grid

The `.glass` utility renders the canonical warm translucent surface:

```css
.glass {
  background: rgba(255, 255, 255, 0.55);
  backdrop-filter: blur(14px);
  border: 1px solid rgba(255, 255, 255, 0.5);
}
.dark .glass {
  background: rgba(40, 37, 34, 0.7);
  border-color: rgba(255, 255, 255, 0.08);
}
```

The `<body>` ships with a fixed dot-grid overlay at 16px × 16px — tiny
ink dots at 0.08 alpha in light, white dots at 0.04 alpha in dark. This
is intentional — it reinforces the sense of a reference grid without
asserting a visible ruler.

### Amber focus ring

Every interactive primitive (button, input, select, toggle, checkbox,
radio, date picker) lands on the same amber halo when focused:

```
border-color: var(--color-amber);
box-shadow: 0 0 0 3px rgba(245, 158, 11, 0.12);
```

The utility class `.ds-focus-ring` bakes this into any arbitrary target.

| Primitive | When to use |
|-----------|-------------|
| `Input` | Single-line text / email / password. `variant="raised" \| "glass"` |
| `NumberField` | Numeric with optional `prefix` / `suffix` (`$`, `d`, `%`). `mono` prop for tabular numerals |

## Conventions

- **Labels** use `.ds-label` (JetBrains Mono 10px uppercase).
- **Primary CTAs** default to `Button variant="filled"` (ink-dark). Reserve
  `variant="amber"` for the single most important action per view (e.g.
  "Advance →" on the detail pipeline bar).
- **Destructive** actions use `variant="danger"` — never a red filled.
- **Mono data** (dates, money, measurements) renders in JetBrains Mono
  or with `tabular-nums` for vertical alignment.
- **Glass** is the default surface. Reach for `bg-cream-2` solid only
  for nested / inset surfaces.
