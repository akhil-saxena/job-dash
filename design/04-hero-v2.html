Hero Section — Rethought

In hero: Company, role, status (editable dropdown), salary, location type, days, pin/archive.
Not in hero: Tags (overview tab), source (overview), applied date (overview), posting URL (overview).
Rule: Everything in the hero is either identity (who) or decision-critical (what stage, how much, where).

3 Hero Variants — Desktop + Mobile

A: Dropdowns + Info Pills
Status and priority as editable dropdowns. Salary, location, days as subtle info pills. Actions right-aligned. Clean separation between editable (dropdowns) and read (pills).
