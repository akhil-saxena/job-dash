## Rules

1. Speed-lines must keep 4u padding from the left edge — never touching.
2. The J always leans right (-14° skew). Never flip the direction.
3. The tail is always amber. Do not recolor with product status colors.
4. Below 32px use the reduced 2-line variant. Below 20px use J only.
