**Step 1 — src/client/hooks/useApplications.ts (per Research Pattern 6):**
- Define `Application` interface matching the API response shape (id, companyName, roleTitle, status, priority, slug, isPinned, isArchived, updatedAt, createdAt, appliedAt, source, locationType, locationCity, salaryMin, salaryMax, salaryCurrency, notes). Note: `updatedAt` and `createdAt` are unix epoch numbers from D1 (not ISO strings). Confirm by checking the API response format.
- `useApplications()`: `useQuery({ queryKey: ['applications'], queryFn })` — fetches `GET /api/applications?limit=100&archived=false`, extracts `json.data.items`
- `useApplicationsByStatus()`: Calls `useApplications()`, groups results into a `Map<string, Application[]>`. Initialize ALL statuses with empty arrays first (per Pitfall 6 — prevent column collapse). Then distribute applications. Return `{ ...query, grouped }`.
- `useCreateApplication()`: `useMutation` calling `POST /api/applications` with JSON body, `onSuccess` invalidates `['applications']` query

**Step 2 — src/client/lib/urgency.ts (per D-16, D-17, D-18, Research Code Example):**
- Export `UrgencyLevel` type: `'interview-today' | 'interview-tomorrow' | 'interview-week' | 'offer-expiring' | 'stale' | 'rejected' | 'normal'`
- `calculateUrgency(app: { status: string; updatedAt: number })`: returns UrgencyLevel
  - If `status === 'rejected'` return `'rejected'`
  - Calculate days since update: `Math.floor((Date.now() / 1000 - app.updatedAt) / 86400)` (updatedAt is unix seconds from D1)
  - If `daysSinceUpdate >= 7` and status not in `['rejected', 'withdrawn', 'accepted']` return `'stale'`
  - Interview/offer urgency will be added in Phase 5/6 when those dates exist
  - Otherwise return `'normal'`
- Export `URGENCY_STYLES: Record<UrgencyLevel, string>`:
  - `'interview-today'`: `'bg-[rgba(245,158,11,0.06)] border border-[rgba(245,158,11,0.15)]'`
  - `'interview-tomorrow'`: same as above
  - `'interview-week'`: `''` (no tint)
  - `'offer-expiring'`: `'bg-[rgba(34,197,94,0.06)] border border-[rgba(34,197,94,0.15)]'`
  - `'stale'`: `'bg-[rgba(239,68,68,0.05)] border border-[rgba(239,68,68,0.12)]'`
  - `'rejected'`: `'opacity-40'` (per D-18)
  - `'normal'`: `''`
- Export helper `getDaysSinceUpdate(updatedAt: number): number`
