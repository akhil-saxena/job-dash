2. In `src/client/hooks/useApplications.ts`, add three optimistic mutation hooks BELOW the existing exports (per D-16, D-17, D-18, UI-05):

   **useUpdateStatus():**
   - `mutationFn`: PATCH `/api/applications/${id}/status` with `{ status }` body
   - `onMutate`: (a) `await queryClient.cancelQueries({ queryKey: ["applications"] })`, (b) snapshot `previousApps = queryClient.getQueryData(["applications"])`, (c) `queryClient.setQueryData(["applications"], old => old?.map(app => app.id === id ? { ...app, status, updatedAt: Math.floor(Date.now() / 1000) } : app))`, (d) return `{ previousApps }`
   - `onError`: rollback `queryClient.setQueryData(["applications"], context.previousApps)`, call `showToast("Failed to update. Reverted.", "error")` (per D-18)
   - `onSettled`: `queryClient.invalidateQueries({ queryKey: ["applications"] })`
   - Note: Import `useToast` from `@/client/components/ui/Toast`

   **useTogglePin():**
   - `mutationFn`: PATCH `/api/applications/${id}/pin` (no body)
   - `onMutate`: cancel queries, snapshot, optimistically toggle `isPinned` (flip 0/1), return context
   - `onError`: rollback + toast "Failed to update. Reverted."
   - `onSettled`: invalidate

   **useToggleArchive():**
   - `mutationFn`: PATCH `/api/applications/${id}/archive` (no body)
   - `onMutate`: cancel queries, snapshot, optimistically toggle `isArchived` (flip 0/1), return context
   - `onError`: rollback + toast "Failed to update. Reverted."
   - `onSettled`: invalidate

   Also add a **useUpdateApplication()** mutation for partial field updates (used by detail page Overview tab):
   - `mutationFn`: PATCH `/api/applications/${id}` with partial body
   - Same optimistic pattern: cancel, snapshot, merge fields, rollback on error
   - `onSettled`: invalidate both `["applications"]` and `["application", slug]`

   Each hook must accept no arguments and return the `useMutation` result. The `mutate` call receives `{ id, status }` or `{ id }` or `{ id, ...fields }`.

4. Create `src/client/hooks/useApplicationDetail.ts` (per D-06, D-12):
   ```typescript
   import { useQuery } from "@tanstack/react-query";
   import type { Application } from "./useApplications";

   interface TimelineEvent {
     id: string;
     applicationId: string;
     eventType: string;
     description: string;
     metadata: string | null;
     occurredAt: string;
   }

   export interface ApplicationDetail extends Application {
     jobPostingUrl: string | null;
     equity: string | null;
     bonus: string | null;
     timeline: TimelineEvent[];
   }

   export function useApplicationBySlug(slug: string) {
     return useQuery({
       queryKey: ["application", slug],
       queryFn: async () => {
         const res = await fetch(`/api/applications/by-slug/${slug}`);
         if (!res.ok) throw new Error("Application not found");
         const json = await res.json();
         return json.data as ApplicationDetail;
       },
       enabled: !!slug,
     });
   }
   ```
