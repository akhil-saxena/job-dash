**1. Create `src/server/services/interview.ts`:**

Follow the exact patterns from `application.ts` service. Import `Database` from `@/server/lib/db`, `NotFoundError` from `@/server/lib/errors`, `eq`, `and`, `asc` from `drizzle-orm`, `nanoid` from `nanoid`, `interviewRound`, `interviewQa`, `application` from `@/db/schema`.

Service functions (all accept `db: Database, userId: string` as first two args for tenant isolation):

**`createRound(db, userId, applicationId, input)`:**
- Verify application exists and belongs to user: `db.select({ id: application.id }).from(application).where(and(eq(application.id, applicationId), eq(application.userId, userId), isNull(application.deletedAt))).get()` -- throw `NotFoundError("Application not found")` if missing
- Compute `sortOrder`: count existing rounds for this application, use `count + 1` (or 0 if first)
- Insert into `interviewRound` with: `id: nanoid()`, `applicationId`, `userId`, all input fields, `sortOrder`, `createdAt: new Date()`, `updatedAt: new Date()`
- Handle `scheduledAt`: if provided as ISO string, convert to `new Date(input.scheduledAt)` for Drizzle timestamp mode
- Return the inserted row via `.returning()`

**`listRounds(db, userId, applicationId)`:**
- Verify application exists + belongs to user (same check as createRound)
- Fetch all rounds: `db.select().from(interviewRound).where(and(eq(interviewRound.applicationId, applicationId), eq(interviewRound.userId, userId))).orderBy(asc(interviewRound.sortOrder), asc(interviewRound.createdAt)).all()`
- For each round, fetch QA pairs: `db.select().from(interviewQa).where(and(eq(interviewQa.roundId, round.id), eq(interviewQa.userId, userId))).orderBy(asc(interviewQa.sortOrder)).all()`
- Return rounds array with `qaPairs` nested: `rounds.map(r => ({ ...r, qaPairs: qaByRound.get(r.id) || [] }))`
- OPTIMIZATION: Fetch all QA pairs for all round IDs in one query using `inArray(interviewQa.roundId, roundIds)`, then group by roundId client-side. Avoid N+1.

**`updateRound(db, userId, roundId, input)`:**
- Verify round exists + belongs to user: `db.select().from(interviewRound).where(and(eq(interviewRound.id, roundId), eq(interviewRound.userId, userId))).get()` -- throw NotFoundError if missing
- PARTIAL UPDATE: only set fields present in `input`. Build `setValues` object from input, always add `updatedAt: new Date()`
- Handle `scheduledAt` string -> Date conversion if present
- `db.update(interviewRound).set(setValues).where(eq(interviewRound.id, roundId)).returning()`
- Return updated row

**`deleteRound(db, userId, roundId)`:**
- Verify round exists + belongs to user (same pattern)
- `db.delete(interviewRound).where(eq(interviewRound.id, roundId))` -- QA pairs cascade-delete via FK
- Return `{ deleted: true }`

**`createQA(db, userId, roundId, input)`:**
- Verify round exists + belongs to user
- Compute sortOrder from count of existing QA pairs for this round
- Insert into `interviewQa` with `id: nanoid()`, `roundId`, `userId`, `question`, `answer` (nullable), `sortOrder`, timestamps
- Return inserted row via `.returning()`

**`updateQA(db, userId, qaId, input)`:**
- Verify QA exists + belongs to user: `db.select().from(interviewQa).where(and(eq(interviewQa.id, qaId), eq(interviewQa.userId, userId))).get()`
- Partial update, always set `updatedAt: new Date()`
- Return updated row

**`deleteQA(db, userId, qaId)`:**
- Verify QA exists + belongs to user
- Delete and return `{ deleted: true }`

**`countRounds(db, userId, applicationId)`:**
- Returns `{ count: number }` -- used by DetailPage for tab badge
- `db.select({ value: count() }).from(interviewRound).where(and(eq(interviewRound.applicationId, applicationId), eq(interviewRound.userId, userId))).get()`
